字号

a、建议下方、链接等部分使用14号字体，标题部分使用16px；

b、建议中文字体为：微软雅黑，数字/英文字体为：Helvetica；

c、web页面内容较多，表单内容复杂，字号建议多为14px，以不同色 值进行层级区分。

应用

h1 .h1 24号字体

h2 .h2 20号字体

h3 .h3 18号字体

h4 .h4 16号字体

h5 .h5 14号字体

20px：适用于首页部分，宣传或警示类文字；

18px：适用于整页面大标题；

16px：适用于整页面部分标题；

14px：适用于大部分的小标题，正文；
